
# Bomber-Man

> A remake for the classic bomber-man video game.
Developed in Javascript using Phaser.js library.


##### Features (complete and un-complete)
- [x] Person VS. Person up to 4 players.
- [x] Mobile support.
- [ ] AI (playing VS. Computer).
- [ ] Multi-player game.
